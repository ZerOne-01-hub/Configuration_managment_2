# Инструмент визуализации графа зависимостей пакетов

Инструмент для анализа и визуализации графа зависимостей пакетов JavaScript (npm). Разработан без использования готовых менеджеров пакетов и библиотек для получения зависимостей.

## Возможности

- Анализ зависимостей npm пакетов
- Построение графа зависимостей с учетом транзитивности
- Визуализация зависимостей в формате ASCII-дерева
- Генерация диаграмм на языке D2
- Фильтрация пакетов по подстроке
- Поддержка тестового режима

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ZerOne-01-hub/Configuration_managment_2.git
cd Configuration_managment_2
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Использование

1. Настройте конфигурационный файл `config.yaml`:
```yaml
package_name: "express"
repository_url: "https://registry.npmjs.org"
test_mode: false
package_version: "latest"
ascii_tree_mode: true
filter_substring: ""
```

2. Запустите приложение:
```bash
python3 main.py
```

Или с указанием другого конфигурационного файла:
```bash
python3 main.py -c path/to/config.yaml
```

## Конфигурация

### Параметры конфигурационного файла

- `package_name` - имя анализируемого пакета
- `repository_url` - URL-адрес репозитория (например, https://registry.npmjs.org)
- `test_mode` - режим работы с тестовым репозиторием (true/false)
- `test_repository_path` - путь к файлу тестового репозитория
- `package_version` - версия пакета (например, "latest", "4.18.2")
- `ascii_tree_mode` - режим вывода зависимостей в формате ASCII-дерева (true/false)
- `filter_substring` - подстрока для фильтрации пакетов
